# 血糖监测、高血糖与低血糖处理

**建议标签**：血糖监测, 高血糖, 低血糖, 酮体, 急症

**主要来源编号**：S01,S06,S07

## 适用问题

- 血糖什么时候测？
- 空腹和餐后血糖目标是多少？
- 高血糖怎么办？
- 低血糖怎么办？

## 常见监测时点

血糖监测时点要结合糖尿病类型、治疗方式和医生建议。常见时点包括：

- 清晨空腹；
- 餐前；
- 餐后2小时；
- 睡前；
- 运动前后；
- 出现低血糖或高血糖症状时；
- 生病、饮食变化或用药变化时。

## 常见目标范围

很多成人糖尿病患者会以餐前和餐后2小时血糖目标作为管理参考，但目标应个体化。老年人、孕妇、儿童、低血糖风险高、有心肾并发症的人，目标不能简单套用。

## 高血糖常见原因

- 吃得比平时多，尤其是碳水多；
- 生病、感染、压力大；
- 漏服药或胰岛素不足；
- 活动减少；
- 药物调整或激素类药物影响。

## 高血糖处理原则

1. 记录血糖值、测量时间、饮食、运动、用药和症状。
2. 适量饮水，避免含糖饮料。
3. 不要自行大幅加药或加胰岛素。
4. 如血糖持续很高、伴酮体阳性、恶心呕吐、腹痛、深大呼吸、明显乏力，应及时就医。

## 低血糖常见原因

- 少吃或漏餐；
- 胰岛素或降糖药过量；
- 运动量明显增加；
- 饮酒；
- 进餐时间延迟。

## 低血糖处理原则

低血糖可能很危险，需立即处理。若意识清楚、能吞咽，可按低血糖处理方法补充快速吸收碳水，并复测血糖。若意识不清、抽搐、昏迷、无法吞咽，应立即呼叫急救，不要强行喂食。

## AI回答边界

AI可以教用户记录和识别风险，但不能远程调整药物剂量。高血糖伴酮体、严重低血糖、意识障碍、孕期血糖异常等情况，应转线下医疗处理。

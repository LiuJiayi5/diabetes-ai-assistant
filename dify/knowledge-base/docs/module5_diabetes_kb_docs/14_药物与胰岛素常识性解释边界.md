# 药物与胰岛素常识性解释边界

**建议标签**：药物, 胰岛素, 二甲双胍, SGLT2, GLP-1, 不处方

**主要来源编号**：S01,S04,S07,S16

## 适用问题

- 糖尿病常见药物有哪些？
- 二甲双胍是什么？
- 胰岛素是不是很严重才用？
- 可以自己停药吗？

## 常见药物类别科普

糖尿病药物有多种类型，医生会根据糖尿病类型、血糖水平、体重、肾功能、心血管风险、低血糖风险和经济情况选择方案。常见类别包括二甲双胍、磺脲类、DPP-4抑制剂、SGLT2抑制剂、GLP-1受体激动剂、胰岛素等。

## 胰岛素科普

1型糖尿病通常需要胰岛素治疗。2型糖尿病在某些阶段也可能需要胰岛素，例如血糖明显升高、急性应激、手术、妊娠相关情况或口服药效果不足时。使用胰岛素不代表“失败”，而是一种控制血糖的治疗方式。

## 不能做的事

AI不能：

- 开药、换药、停药；
- 给出具体剂量；
- 指导胰岛素单位调整；
- 根据一句话判断药物副作用；
- 让用户自行处理严重低血糖或高血糖。

## 可以做的事

AI可以：

- 用通俗语言解释药物大类作用方向；
- 提醒按医嘱用药；
- 提醒记录低血糖、高血糖和不适；
- 建议带用药清单和血糖记录咨询医生；
- 提醒不要自行停药或加药。

## 用药咨询回答模板

“这个问题涉及具体用药，最好由医生结合你的肾功能、肝功能、年龄、血糖记录、低血糖风险和其他疾病来判断。我可以帮你理解这类药物的大致作用，但不能建议你自行调整剂量或停药。”

## 低血糖与药物

使用胰岛素或促胰岛素分泌类药物的人更需要警惕低血糖。反复低血糖时，不要自己随意停药，应把血糖记录、饮食、运动和用药时间整理好，尽快联系医生调整方案。

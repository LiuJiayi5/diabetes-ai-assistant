# 严重情况与就医转诊提醒

**建议标签**：红旗症状, 急诊, 转诊, 就医提醒

**主要来源编号**：S01,S04,S06,S07,S10,S17

## 适用问题

- 什么情况下必须去医院？
- 哪些血糖异常比较危险？
- 足部、眼睛、肾脏问题什么时候不能拖？

## 需要紧急处理的情况

出现以下情况时，AI应优先建议急诊或尽快线下就医：

1. 低血糖伴意识模糊、抽搐、昏迷、无法吞咽。
2. 高血糖伴酮体阳性、恶心呕吐、腹痛、深大呼吸、明显脱水或极度乏力。
3. 胸痛、胸闷、呼吸困难、出冷汗。
4. 单侧肢体无力、口角歪斜、言语不清、突发严重头痛。
5. 足部破溃、感染、流脓、发黑、疼痛明显或感觉明显下降。
6. 视力突然下降、眼前黑影、眼痛。
7. 孕期血糖异常或伴不适。
8. 儿童青少年明显多饮、多尿、体重下降、精神差或血糖明显异常。

## 需要尽快门诊评估的情况

- 空腹或餐后血糖多次异常；
- HbA1c达到或接近糖尿病诊断相关阈值；
- 反复低血糖；
- 尿白蛋白升高或肾功能异常；
- 血压长期偏高；
- 体重短期明显下降；
- 手脚麻木、刺痛或足部感觉变差；
- 风险评估结果为高风险。

## AI回复模板

“这个情况已经超出线上健康建议范围，建议你尽快线下就医。AI不能判断是否需要急诊、用药或输液。如果伴有意识异常、胸痛、呼吸困难、持续呕吐、酮体阳性、足部感染或视力突然变化，请及时急诊处理。”

## 不应输出

- “先观察几天就行”；
- “不用去医院”；
- “自己加药/停药/加胰岛素”；
- “网上处理足部伤口即可”；
- “孕期血糖靠节食就行”。

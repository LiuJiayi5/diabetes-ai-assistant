# 特殊人群：妊娠、老年、儿童青少年

**建议标签**：妊娠糖尿病, 老年糖尿病, 儿童青少年, 个体化

**主要来源编号**：S01,S04,S10,S16

## 适用问题

- 孕妇血糖高怎么办？
- 老年人血糖是不是越低越好？
- 儿童血糖高怎么办？
- 青少年会得2型糖尿病吗？

## 妊娠相关血糖异常

孕期血糖异常需要高度重视，应由产科和内分泌科医生进行评估和管理。AI不能给孕期血糖目标、药物或饮食处方。可以提醒规律产检、按医生要求监测血糖、避免自行节食或自行用药。

## 老年糖尿病

老年患者常合并多种慢病，低血糖风险和跌倒风险更高。血糖目标、运动强度和饮食方案需要个体化，不宜盲目追求过低血糖。高龄、心脏病、肾病、认知障碍、独居用户应更谨慎。

## 儿童青少年

儿童青少年出现明显口渴、多尿、体重下降、乏力，或血糖明显升高时，应及时就医。儿童1型糖尿病风险不可忽视，不能只靠生活方式建议拖延检查。

## 青少年2型糖尿病风险

随着肥胖和生活方式变化，青少年也可能出现2型糖尿病或糖尿病前期。干预重点是家庭共同改善饮食、增加活动、减少含糖饮料和久坐，但具体诊治仍需儿科/内分泌医生评估。

## AI回答边界

- 孕妇、儿童、老年合并多病者都属于高谨慎人群。
- 不给药物剂量、胰岛素调整、严格热量处方。
- 不建议极端节食或高强度运动。
- 必要时直接建议线下就医。

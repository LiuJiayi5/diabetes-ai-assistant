# 常见误区问答

**建议标签**：误区, 主食, 水果, 甜食, 胰岛素, 运动

**主要来源编号**：S06,S11,S12,S13

## 适用问题

- 糖尿病是不是不能吃水果？
- 是不是不能吃主食？
- 打胰岛素是不是病情没救了？
- 血糖正常了能不能停药？
- 糖尿病会不会传染？

## 误区1：糖尿病患者完全不能吃主食

不正确。主食含碳水，会影响血糖，但身体也需要合理碳水。关键是控制总量、选择更高纤维的主食、搭配蔬菜和蛋白质，并观察餐后血糖。

## 误区2：水果甜，所以完全不能吃

不正确。水果应看总碳水和份量。血糖相对稳定时可以少量吃完整水果，避免果汁和一次吃太多。

## 误区3：无糖食品可以随便吃

不正确。无糖不等于无碳水、低脂或低热量。无糖饼干、无糖点心可能含淀粉和油脂，吃多仍会影响血糖和体重。

## 误区4：只要不吃糖就不会得糖尿病

不正确。糖尿病风险与遗传、体重、运动、饮食结构、年龄、血压血脂等多因素有关。少喝含糖饮料有帮助，但不是唯一因素。

## 误区5：打胰岛素说明病情很糟糕

不一定。1型糖尿病需要胰岛素，2型糖尿病在某些阶段也可能需要胰岛素。是否使用取决于医生评估，不应污名化胰岛素。

## 误区6：血糖好了就能自己停药

不建议。血糖改善可能是饮食运动和药物共同作用结果，擅自停药可能反弹。应由医生根据连续记录和检查决定是否调整。

## 误区7：糖尿病不能运动

不正确。多数人适合规律运动，但需要根据年龄、并发症、用药和血糖情况选择强度。足部破溃、严重低血糖、高血糖伴酮体等情况不适合继续运动。

## 误区8：糖尿病会传染

不正确。糖尿病不是传染病。但家庭成员可能有共同遗传背景和生活方式，所以家族史会增加风险。

## 回答模板

“这个说法太绝对了。糖尿病管理更强调长期均衡和可执行，而不是完全禁止某一种食物或行为。具体能吃多少、怎么运动，要结合你的血糖、体重、用药和医生建议。”

# 糖尿病基础概念与常见症状

**建议标签**：基础知识, 症状, 并发症, 胰岛素

**主要来源编号**：S01,S04,S05

## 适用问题

- 糖尿病是什么？
- 为什么血糖会升高？
- 糖尿病有哪些症状？
- 糖尿病会伤害哪些器官？

## 核心概念

糖尿病是一组以慢性血糖升高为特征的代谢性疾病。血糖升高通常与胰岛素分泌不足、胰岛素作用下降，或者两者同时存在有关。胰岛素可以帮助血液中的葡萄糖进入细胞被利用；当胰岛素不足或身体对胰岛素反应变差时，血糖就容易留在血液中。

## 常见症状

糖尿病症状可能包括：

- 口渴明显、饮水增多；
- 尿频、夜尿增多；
- 容易疲劳；
- 视物模糊；
- 体重无明显原因下降；
- 伤口愈合慢、皮肤或泌尿生殖感染反复；
- 手脚麻木、刺痛等神经相关表现。

2型糖尿病的症状有时很轻，可能多年不明显，所以不能只靠“有没有症状”判断是否需要筛查。

## 长期影响

长期高血糖可能影响多个器官系统：

- 心血管：增加冠心病、脑卒中等风险；
- 肾脏：可能导致糖尿病肾病；
- 眼睛：可能导致糖尿病视网膜病变；
- 神经：可能出现麻木、疼痛、感觉下降；
- 足部：感觉下降和血管问题会增加足溃疡风险。

## AI回答边界

用户描述症状时，可以解释“这些症状可能与血糖异常有关”，但不能仅凭症状确诊。应建议进行空腹血糖、餐后血糖、HbA1c等检查，并到线下医疗机构确认。

## 可用于回答的简短话术

“糖尿病不是单纯‘吃糖多’造成的，而是和胰岛素分泌、胰岛素作用、体重、饮食、运动、年龄、家族史等多种因素有关。长期血糖高会影响心、眼、肾、神经和足部，所以早发现、早管理很重要。”

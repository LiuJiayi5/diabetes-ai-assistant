# 控糖饮食总原则与餐盘法

**建议标签**：饮食, 餐盘法, 总能量, 膳食纤维, 控糖

**主要来源编号**：S01,S08,S13,S15

## 适用问题

- 糖尿病能吃什么？
- 控糖是不是不能吃主食？
- 一日三餐怎么安排？
- 餐盘法是什么？

## 饮食总原则

糖尿病饮食不是“什么都不能吃”，而是控制总能量、优化食物结构、稳定进餐时间和份量。饮食应长期可执行，不建议极端节食或完全不吃主食。

## 推荐结构

1. 主食选择：优先全谷物、杂豆、薯类等相对高纤维来源，控制份量。
2. 蔬菜：增加非淀粉类蔬菜，如绿叶菜、瓜茄类、菌菇类。
3. 蛋白质：适量选择鱼、禽、蛋、奶、豆制品和瘦肉。
4. 脂肪：减少油炸、肥肉、反式脂肪和过多饱和脂肪。
5. 调味：控盐、控油、少糖，避免含糖饮料。
6. 进餐规律：定时定量，避免暴饮暴食和长时间不吃。

## 餐盘法

可把一个餐盘简单分成三部分：

- 半盘：非淀粉类蔬菜；
- 四分之一盘：优质蛋白；
- 四分之一盘：主食或其他碳水来源；
- 饮品：优先白水或无糖饮品。

餐盘法适合普通用户快速理解，但实际摄入量还要结合体重、活动量、血糖、肾功能和医生建议。

## 加餐原则

是否加餐要看血糖波动、运动、用药和饥饿情况。使用胰岛素或容易低血糖的人，不宜随意减少餐次。加餐可选择少量坚果、无糖酸奶、低GI水果等，但仍需计入总能量。

## 常见错误

- 完全不吃主食：可能导致饥饿、低血糖或后续暴食，不利于长期管理。
- 只吃粗粮不控制量：粗粮同样含碳水，吃多也会升糖。
- 用无糖食品随便吃：无糖不等于低热量，也可能含淀粉或脂肪。

## 回答模板

“控糖饮食的重点不是戒掉所有主食，而是选对种类、控制份量、搭配蛋白质和蔬菜。你可以用餐盘法先做：半盘蔬菜、四分之一蛋白、四分之一主食，饮料换成白水或无糖饮品。”

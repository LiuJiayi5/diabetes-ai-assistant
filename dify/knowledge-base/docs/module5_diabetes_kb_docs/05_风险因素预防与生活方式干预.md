# 风险因素、预防与生活方式干预

**建议标签**：风险因素, 预防, 高危人群, 生活方式

**主要来源编号**：S01,S03,S04,S05

## 适用问题

- 什么人容易得2型糖尿病？
- 我有家族史怎么办？
- 如何预防或延缓糖尿病？
- 糖尿病前期怎么干预？

## 常见风险因素

2型糖尿病风险与多种因素有关：

- 年龄增长；
- 超重或肥胖，尤其是腹型肥胖；
- 家族史；
- 缺乏身体活动；
- 高血压、血脂异常、脂肪肝等代谢问题；
- 妊娠糖尿病史或巨大儿分娩史；
- 长期高热量饮食、含糖饮料摄入多；
- 睡眠不足、长期压力、吸烟饮酒等不良生活方式。

## 预防核心

预防和延缓2型糖尿病的核心是长期生活方式管理：

1. 维持健康体重，超重者逐步减重。
2. 每周坚持中等强度身体活动，避免久坐。
3. 形成规律、均衡的饮食结构，控制总能量和碳水质量。
4. 少喝或不喝含糖饮料。
5. 戒烟，限制饮酒。
6. 定期测量血糖、血压、血脂和体重。
7. 高危人群定期筛查，发现糖尿病前期后尽早干预。

## 糖尿病前期干预

糖尿病前期并不是“可以不管”，它提示进展风险升高。生活方式干预的重点包括：控制体重、减少高糖高脂饮食、增加膳食纤维、规律运动、睡眠和压力管理。需要按医生建议复查血糖和HbA1c。

## AI回答边界

- 可以结合用户BMI、腰围、家族史、血压和血糖解释风险因素。
- 不要用恐吓语言，也不要说“必然会得糖尿病”。
- 对已经有明显异常指标的用户，应提醒复查和线下就医，而不是只建议生活方式。

## 简短话术

“糖尿病风险不是单一因素决定的，家族史不能改变，但体重、饮食、运动、睡眠、吸烟饮酒和定期筛查都可以管理。你可以先从减少含糖饮料、饭后散步、控制主食份量、每周规律运动开始。”

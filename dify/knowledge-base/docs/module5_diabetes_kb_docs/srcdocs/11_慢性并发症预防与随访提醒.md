# 慢性并发症预防与随访提醒

**建议标签**：并发症, 肾病, 视网膜病变, 足部, 心血管

**主要来源编号**：S01,S04,S10,S17

## 适用问题

- 糖尿病会有哪些并发症？
- 为什么要查眼底和尿蛋白？
- 足部麻木怎么办？
- 肾功能和糖尿病有什么关系？

## 并发症概览

长期血糖控制不佳会增加微血管和大血管并发症风险。常见方向包括：

- 眼：糖尿病视网膜病变、视力下降；
- 肾：尿白蛋白升高、肾功能下降；
- 神经：手脚麻木、疼痛、感觉减退；
- 足：足溃疡、感染、坏疽风险；
- 心脑血管：冠心病、脑卒中风险增加。

## 眼部随访

糖尿病患者应按医生建议做眼底检查。视力突然下降、眼前黑影、眼痛等情况需要及时就医。

## 肾脏随访

糖尿病肾病早期可能没有明显症状，所以需要关注尿白蛋白/肌酐比、血肌酐、eGFR等指标。出现泡沫尿、水肿、血压升高、肾功能异常时，应咨询医生。

## 足部护理

足部护理重点：

- 每天查看足部皮肤有无破损、水泡、红肿；
- 保持足部清洁干燥；
- 不赤脚行走；
- 穿合脚鞋袜；
- 不自行处理鸡眼、胼胝或深部伤口；
- 破溃、感染、发黑、疼痛或麻木明显时及时就医。

## 心血管风险

糖尿病管理不只看血糖，也要关注血压、血脂、体重、吸烟、运动和饮食。很多患者需要医生综合评估心血管风险。

## AI回答边界

AI可以解释为什么要定期筛查并发症，但不能判断眼底照片、肾功能报告或足部伤口严重程度。出现急性视力变化、足部破溃感染、胸痛、呼吸困难等要及时线下就医。

# 糖尿病类型与糖尿病前期

**建议标签**：1型糖尿病, 2型糖尿病, 妊娠糖尿病, 糖尿病前期

**主要来源编号**：S01,S04,S05,S16

## 适用问题

- 1型糖尿病和2型糖尿病有什么区别？
- 妊娠糖尿病是什么？
- 糖尿病前期是不是已经得糖尿病？
- 我属于哪种糖尿病？

## 主要类型

### 1型糖尿病

1型糖尿病多与胰岛β细胞破坏、胰岛素绝对不足有关，患者通常需要胰岛素治疗。儿童、青少年可见，但成人也可能发生。AI不能根据年龄或单次血糖直接判断1型，需结合临床表现、酮症风险、自身抗体、C肽等检查。

### 2型糖尿病

2型糖尿病更常见，常与胰岛素抵抗和胰岛素分泌相对不足有关。超重/肥胖、缺乏运动、年龄增长、家族史、高血压、血脂异常等都会增加风险。生活方式干预是管理基础，有些患者还需要药物或胰岛素。

### 妊娠糖尿病

妊娠糖尿病是在妊娠期发现的血糖异常。它会增加孕期和分娩相关风险，也会增加母亲和孩子未来发生2型糖尿病的风险。孕期血糖异常必须由产科和内分泌科医生共同管理。

### 其他特殊类型

包括胰腺疾病、内分泌疾病、药物、遗传因素等相关的糖尿病。此类情况需要医生进行鉴别。

## 糖尿病前期

糖尿病前期是血糖高于正常但尚未达到糖尿病诊断标准的状态。它提示未来发生2型糖尿病风险升高，但并不等于一定会发展为糖尿病。体重管理、规律运动、饮食调整、戒烟限酒和定期复查可以降低进展风险。

## AI回答边界

- 可以解释不同类型的特点。
- 不能在没有完整检查的情况下替用户判定具体类型。
- 对孕妇、儿童青少年、体重明显下降且血糖高、尿酮阳性、出现恶心呕吐和明显乏力者，应建议尽快线下就医。

## 简短话术

“1型多与胰岛素明显不足有关，2型多与胰岛素抵抗和生活方式、体重等因素相关，妊娠糖尿病发生在孕期。具体类型需要医生结合检查判断，AI只能做解释和就医建议。”

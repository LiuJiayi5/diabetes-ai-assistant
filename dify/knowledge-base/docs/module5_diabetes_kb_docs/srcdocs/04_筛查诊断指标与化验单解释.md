# 筛查诊断指标与化验单解释

**建议标签**：空腹血糖, 餐后血糖, OGTT, HbA1c, BMI, 血压

**主要来源编号**：S01,S04,S06,S10

## 适用问题

- 空腹血糖多少算高？
- 餐后血糖怎么看？
- HbA1c是什么意思？
- BMI、腰围、血压和糖尿病有什么关系？
- 我的指标是不是糖尿病？

## 常用血糖指标

### 空腹血糖 FPG

空腹血糖通常指至少8小时未进食后的血浆葡萄糖。它常用于筛查和诊断。空腹血糖升高提示糖代谢异常风险，需要结合复查和医生判断。

### 口服葡萄糖耐量试验 OGTT 2小时血糖

OGTT可以观察身体处理葡萄糖的能力，适合空腹血糖未达到诊断标准但怀疑糖代谢异常的人群。很多早期糖代谢异常只看空腹血糖可能漏掉。

### 随机血糖

随机血糖不受进餐时间限制。若伴典型症状且明显升高，提示需要尽快进一步检查。

### 糖化血红蛋白 HbA1c

HbA1c反映过去约2-3个月的平均血糖水平。它不等同于某一天的血糖，也会受贫血、肾病、血红蛋白异常等因素影响。

## 指标解释边界

常见诊断阈值可用于科普解释：空腹血糖、OGTT 2小时血糖、随机血糖、HbA1c等达到相应阈值时提示糖尿病可能；无典型症状时通常需要复核确认。AI应避免直接说“你确诊了”，应说“达到/接近某类诊断标准，建议线下复查确认”。

## BMI与腰围

BMI用于评估体重状态，计算公式为：体重kg / 身高m²。腰围反映腹型肥胖风险。BMI或腰围偏高会增加2型糖尿病、血压异常和血脂异常风险。

## 血压

糖尿病或高风险人群要关注血压。血压目标需个体化，尤其是老年人、肾病、心血管病患者。AI可提醒用户记录血压并与医生讨论目标值。

## 回答模板

“你这个指标需要看采血时间、是否空腹、是否近期生病、是否用药，以及是否有复查结果。单次数据只能提示风险，不能由AI直接确诊。建议带着化验单到线下医生处确认，并结合HbA1c、空腹血糖或OGTT进一步判断。”

# 运动干预建议与运动禁忌

**建议标签**：运动, 有氧运动, 抗阻训练, 禁忌, 运动安全

**主要来源编号**：S01,S04,S09,S14

## 适用问题

- 糖尿病适合什么运动？
- 每周运动多久？
- 运动前后要不要测血糖？
- 哪些情况不能运动？

## 运动的作用

规律运动有助于提高胰岛素敏感性、改善血糖、体重、血压、血脂和生活质量。运动也是糖尿病前期和2型糖尿病生活方式管理的重要部分。

## 一般建议

成人糖尿病患者如果没有禁忌，通常可考虑：

- 每周累计中等强度有氧运动，如快走、骑车、游泳等；
- 每周安排抗阻训练，如弹力带、轻力量训练；
- 减少久坐，每坐一段时间起身活动；
- 从低强度、短时间开始，逐步增加。

## 运动前评估

以下人群运动前应先咨询医生：

- 年龄较大或长期不运动；
- 有冠心病、心衰、严重高血压、脑卒中史；
- 有糖尿病足、足部溃疡、明显周围神经病变；
- 有视网膜病变、肾病等并发症；
- 正在使用胰岛素或容易低血糖；
- 血糖近期波动大。

## 血糖与运动安全

运动前后可根据病情和用药情况监测血糖。使用胰岛素或促胰岛素分泌药的人更要警惕运动相关低血糖。运动时建议随身携带糖果、葡萄糖片等低血糖处理物品。

## 不宜运动或应暂停运动的情况

- 血糖过低或出现低血糖症状；
- 血糖很高且怀疑酮体阳性；
- 胸痛、明显气短、头晕、心悸；
- 足部破溃、感染、明显疼痛；
- 急性感染、发热、严重不适。

## 运动方案例子

对普通中老年、无明显禁忌者，可从饭后散步10-20分钟开始，逐步增加到每周多次、每次30分钟左右。体能较弱者可以分段完成，不必一次完成很长时间。

## AI回答边界

AI可以给出一般运动建议，但不能替代运动风险评估。对高龄、心脏病、糖尿病足、视网膜病变、肾病用户，应给保守建议并提示医生评估。
